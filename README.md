# Customer Segmentation Dashboard using Power BI

## Project Overview

This project focuses on customer segmentation using Power BI. Customers were grouped into different segments based on behavioral and demographic attributes through Power BI's clustering feature.

The dashboard provides insights into customer distribution, premium contribution, policy preferences, communication channels, gender distribution, and occupation trends.

## Objectives

* Segment customers into meaningful groups
* Identify high-value customer segments
* Analyze customer demographics and behavior
* Support targeted marketing and customer retention strategies

## Dataset Features

* Customer ID
* Age
* Gender
* Occupation
* Income Level
* Coverage Amount
* Premium Amount
* Policy Type
* Preferred Communication Channel
* Preferred Language
* Marital Status

## Dashboard Components

### KPI Cards

* Total Customers
* Total Premium
* Average Income
* Average Coverage

### Visualizations

* Customer Clustering Scatter Plot
* Customer Distribution by Segment
* Premium Amount by Segment
* Gender Analysis
* Policy Type Distribution
* Preferred Communication Channel Analysis
* Occupation Analysis

### Interactive Filters

* Gender
* Policy Type
* Occupation
* Customer Segment

## Key Insights

* Cluster1 represents the highest-value customer segment.
* Cluster2 contributes the lowest premium revenue.
* Group policies are the most preferred policy type.
* In-Person communication is the most preferred communication channel.
* Salespersons and Entrepreneurs form the largest customer groups.

## Tools Used

* Power BI Desktop
* Microsoft Excel

## Outcome

The dashboard enables data-driven decision-making by identifying customer segments and providing actionable business insights for marketing and customer retention strategies.
